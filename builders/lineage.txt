*Cargo
  2014
  SY:2
  Bundler
  # the model modern tool: builds AND manages dependencies (crates.io);
  # designed by the Bundler authors, predates Rust 1.0
  # not the first build+package tool (Maven 2004, Cabal 2005 predate it),
  # but the influential ergonomic template (Cargo.toml + lockfile + semver)
  # that Go modules, Swift PM, Mix, Dub later followed
