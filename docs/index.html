Principia Softwarica

Fundamental Literate System Programs

Principia Softwarica is a series of books explaining how things work in a computer by describing with full details all the source code of all the essential programs used by a programmer.

Among those essential programs are the kernel, the shell, the windowing system, the compiler, the linker, the editor, and the debugger. Each program is covered by a separate book.

The books not only describe the implementations of essential programs — they are the implementations. Each program comes from a literate program, a document containing both source code and documentation where the code is organized and presented to facilitate comprehension. The actual code and the book are both derived automatically from this literate program.

The Books

Core System

  • Kernel — process management, memory, traps, device drivers (ARM and x86)
  • Shell — the rc shell: parsing, tree-walking execution, pipes, variables
  • Windowing System — rio: window management, graphics, mouse/keyboard

Development Toolchain

  • Compiler — the C compiler: lexing, parsing, code generation (ARM and x86)
  • Linker — object files, symbol resolution, relocation
  • Assembler — two-pass assembly, instruction encoding (ARM and x86)

Developer Tools

  • Builder — mk, the Plan 9 build system (like make, but better)
  • Debugger — acid: process inspection, breakpoints, stack traces
  • Editor — ed and sam: line and structural editing
  • Profiler — execution profiling
  • Compiler Generator — yacc for C

Libraries

  • Core Library — libc: strings, memory, formatted I/O, system calls
  • Graphics Library — libdraw: drawing primitives, fonts, images
  • GUI Library — libpanel: widgets, menus, text fields
  • Math Library — floating point, arbitrary precision
  • Networking Library — IP, TCP, UDP, DNS
  • Security Library — authentication, encryption

Graphics

  • Windowing System — rio: multiplexing the screen, mouse, keyboard
  • Browser — mothra: HTML rendering, HTTP
  • Games — a few classic games

Networking

  • Network Applications — HTTP, FTP, SMTP, DNS tools

Utilities

  • Utilities — cat, grep, sed, awk, sort, and many more

The Platform

All the programs in Principia Softwarica come from Plan 9 from Bell Labs, an operating system designed by Ken Thompson, Rob Pike, Dave Presotto, and Phil Winterbottom at Bell Labs as the successor to Unix.

Plan 9 was chosen because its code is remarkably elegant and small. You can realistically understand the entire operating system. The code is written in a clean, consistent C style, and the system design follows a few powerful ideas (everything is a file, per-process namespaces, network transparency) applied uniformly.

Source Code

The source code for the Plan 9 fork used in Principia Softwarica is available on GitHub:

  • github.com/aryx/principia-softwarica — the Plan 9 code and literate programs
  • github.com/aryx/goken9cc — cross-compiler to build Plan 9 from Linux/macOS/Windows

OCaml Ports

Some Plan 9 programs have been rewritten in OCaml, with their own literate programming books:

  • xix — OCaml ports of mk (omk), rc (orc), ed (oed), and the assembler
  • codemap — semantic code visualizer (treemap)
  • efuns — Emacs clone in OCaml with GTK/Cairo
  • syncweb — literate programming tool (Noweb)

About
